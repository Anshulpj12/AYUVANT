# 📁 AYUVANT Context Directory

This folder contains detailed documentation for every feature of the AYUVANT Smart Pharmacy System.

## Structure

```
context/
├── README.md                    # This file - Overview
├── project-overview.md          # Full project vision & goals
├── part1-core-platform/         # Part 1: Core Pharmacy Platform
│   ├── overview.md              # Part 1 summary
│   └── feature-batch-entry.md   # Medicine Batch Entry feature
├── part2-ai-intelligence/       # Part 2: AI & Smart Supply Chain
│   └── overview.md              # Part 2 summary
└── design-system.md             # Design system documentation
```

## How This Works
- Every new feature gets its own documentation file
- Each feature file contains: description, user flow, data model, API endpoints, UI screens
- When a feature is built, its doc is updated with implementation details
