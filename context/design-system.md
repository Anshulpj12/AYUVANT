## Design Principles

### Glassmorphism
- Frosted glass panels with 20% opacity white/cyan
- 12px backdrop blur
- Ultra-thin (1px) semi-transparent borders

### Neon Glow Effects
- Electric cyan glow on interactive elements
- Purple glow for secondary actions
- Green glow for success states

### Component Styles
- **Buttons**: Gradient-filled (Cyan → Purple) with outer glow on hover
- **Input Fields**: Dark background with glowing cyan border on focus
- **Cards**: Frosted glass background, 1px border, holographic gradient stripe
- **Status Badges**: Space Grotesk font with colored glow indicators
- **Data Tables**: Alternating dark rows with cyan dividers

### Roundness
- Components use `ROUND_TWELVE` (12px border-radius)

### Color Mode
- Dark mode with Vibrant color variant
